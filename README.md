# Visual Studio Code
_Installation_

https://pimylifeup.com/raspberry-pi-visual-studio-code/

## Install this version
```
sudo apt-get install code-oss=1.29.0-1539702286
```

## Lock the version
```
sudo apt-mark hold code-oss
```

## Unlock the version
```
sudo apt-mark unhold code-oss
```
